# Sobre o Projeto
Software para controle e gerenciamento de transações em bitcoins e smarth contracts de uma empresa fictícia de imóveis, projeto da faculdade do terceiro semestre.

## Tecnologias utilizadas
- C#
- Windowns Forms
- SQL Server
